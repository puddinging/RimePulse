图表重设计方案

解决原版折线图「6 条线做 0–100 归一化后相互叠加 + X 轴标签溢出」的拥挤问题。采用 主图表 + Small multiples 的数据新闻式布局。

关键改动

  • 折线 → 柱状图：日数据本质离散，柱高直接对应量级；峰值柱高亮，今日柱描边。
  • 单指标主图：头部 tab 切换「字数 / 中文 / 英文 / 速度 / 提交」，避免多线叠加。
  • Small multiples：5 个指标每行一条 sparkline，各自独立 y 轴，右侧显示累计值 —— 一眼看齐所有趋势形状。
  • X 轴仅 3 个标签（首/中/末），取代原来每柱一标签。
  • 周期切换：7D / 30D / ALL，快速从全景进入近期。
  • 图例并入 sparkline 行：省去 3×2 图例方阵。

视觉语言

  • 深色 macOS 原生面板 + hairline 分割，保持系统一致性
  • Digital palette（图表限定，沿 #00a5f7 调性）：天蓝（速度）翡翠（中文）紫罗兰（英文）玫瑰粉（时长）琥珀黄（提交）
  • 数字用 tabular-nums + SF Rounded 对齐
  • 组件最小高度下沉，让信息密度提升 ≈30%

落地映射到 SwiftUI

  • 主柱状图 → Chart { BarMark }，.chartXAxis 只保留首/中/末 3 个 label
  • Sparkline → Chart { LineMark } 小尺寸，隐藏双轴
  • 周期 segment → Picker(.segmented)；tab 行可用 HStack + 下划线动画
  • 当前 TrendMetric / TrendGranularity 逻辑可直接复用

交互预览

  • Hover 主图柱子 → tooltip 显示日期 + 精确数值
  • 点击 small multiples 的某一行 → 主图切换到该指标
  • 点击 7D / 30D / ALL → 自动切换按天/周/月粒度
