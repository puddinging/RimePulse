## Project Overview

RimePulse is a macOS menu bar app that displays real-time typing statistics from the [Rime input method](https://rime.im). It reads JSON stats files generated by a Rime Lua plugin and shows today's character count, typing speed, peak speed, 7-day history, and cumulative totals.

- **Platform**: macOS 14 (Sonoma)+
- **Language**: Swift 5.10 / SwiftUI
- **Build system**: Swift Package Manager
- **Bundle ID**: `com.jiefeng.rimepulse`

## Build Commands

```bash
make build          # Debug build (swift build)
make run            # Debug build + run
make release        # Release build
make clean          # swift package clean
make install        # Release build → /usr/local/bin/RimePulse
make bundle         # Release build → ~/Applications/RimePulse.app
```

No tests or linter are currently configured.

## Architecture

**Entry point**: `RimeStatsApp` (`@main`) uses `MenuBarExtra` with `.window` style to render the menu bar popover. `AppDelegate` sets `.accessory` activation policy (no dock icon) and prevents automatic termination.

**Data flow**: `StatsReader` (@Observable, @MainActor) → reads JSON files → publishes `today: TypingStats?` and `history: [TypingStats]` → SwiftUI views react via Observation.

**File watching strategy** in `StatsReader`:
1. Try `DispatchSource` on the today file (`O_EVTONLY`, events: write/rename/delete)
2. If the file doesn't exist yet, fall back to watching the directory for file creation
3. On delete/rename (Lua plugin may recreate the file), automatically restart watching with a short delay
4. Directory watching uses exponential backoff retry (up to 30s) if the directory can't be opened

**View hierarchy**:
- `StatusBarLabel` — menu bar text: char count, CPM, peak CPM (monospaced, emoji-prefixed)
- `StatsPanel` — popover container: today section + history + totals + launch-at-login toggle + quit
  - `TodayStatsView` — big char count, CJK/ASCII breakdown, 4 metric cards (speed, peak, active time, commits), new words
  - `HistoryListView` / `HistoryRowView` — last 7 days in compact rows
  - `TotalStatsView` — aggregated chars, duration, commits across all visible records

**Data sources** (read from `~/Library/Rime` by default, configurable via `~/.config/rimestats/config.json`):
- `typing_stats_today.json` — single JSON object, today's live stats
- `typing_stats.jsonl` — one JSON line per day, historical records

**Model**: `TypingStats` uses `snake_case` CodingKeys to map from JSON. All fields are required (no optionals). `id` is derived from `date`.
